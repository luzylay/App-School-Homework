# Historial de Cambios (Changelog) 📜

Todos los cambios notables en este proyecto serán documentados en este archivo.

## [1.0.0] - 2024-06-21
### Añadido
- Versión inicial de la aplicación AppColegioClass.
- Gestión de Alumnos, Docentes, Cursos y Menús.
- Integración con Cloudinary para fotos.
- Base de datos local con Room.
- Interfaz moderna con Jetpack Compose.
- **Barra de navegación inferior con desplazamiento horizontal** para mejor accesibilidad.

### Seguridad
- Documentación de seguridad para configuración de APIs externas.
- Configuración de `.gitignore` para proteger archivos sensibles.
