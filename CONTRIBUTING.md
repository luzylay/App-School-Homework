# Guía para Contribuir en AppColegioClass 🤝

¡Gracias por tu interés en mejorar este proyecto! Para mantener el orden, sigue estas pequeñas pautas:

## ¿Cómo puedo ayudar?
- **Reportando errores:** Si algo no funciona, abre un "Issue" detallando el problema.
- **Sugiriendo mejoras:** ¿Tienes una idea para una nueva función? ¡Cuéntanoslo!
- **Enviando código:** Si quieres programar una mejora:
    1. Haz un Fork del proyecto.
    2. Crea una rama para tu mejora (`git checkout -b mejora/NuevaFuncion`).
    3. Haz tus cambios y un commit descriptivo.
    4. Envía un Pull Request.

## Estilo de Código
- Intentamos seguir las [guías oficiales de Kotlin](https://kotlinlang.org/docs/coding-conventions.html).
- Mantén las funciones de Jetpack Compose pequeñas y reutilizables.

¡Toda ayuda es bienvenida para mejorar la educación!
